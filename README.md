# 俺はneGconをForza Horizon 6でつかいたいんじゃ for RP2040<BR>(neGconXboxPICO)

オリジナルは[neGcon2SwitchPICO](https://github.com/v9938/neGcon2SwitchPICO)です。

これのファームウェアを、Xbox 360 Controller互換になるよう書き換えたものです。

## Firmware書き込み方法

デバイスのB(BOOTSEL)ボタンを押しながら、デバイスのリセットボタンを押すか、USBケーブルを接続します。
PC側にUSBメモリとして現れるドライブに、neGcon2Xbox_pico_*.uf2をコピーします。

## メタモード
デジタルボタンを押していない状態でSTARTボタンを押すと、メタモードに入ります。

- そのままSTARTボタンを離すと、メニューボタンをタップしたものとして扱います。
- STARTボタンを押している間、以下のアサインになります。

|neGconボタン | Xboxコントローラ | 備考 |
| :- |  :- |  :- |
| ねじり (アナログ) | 右アナログスティックのX/Y | ※2 |
| I (アナログ) |  | ねじり(アナログ)の機能切り替え。 |
| II (アナログ) | 左アナログスティックのY減少側 | ※3 |
| L (アナログ) | 左アナログスティックのY増加側 | ※3 |
| 十字キー上 | Yボタン | |
| 十字キー下 | XBOXボタン | |
| 十字キー左 | LBボタン | |
| 十字キー右 | RBボタン | |
| R (デジタル) | ※1 | |
| A (デジタル) | 右スティック押し込み | |
| B (デジタル) | 左スティック押し込み | |

- ※1: メニュー+Rボタン。Steamのコントローラ設定モードで、ここに好きな操作をアサインできます。
- ※2: I (アナログ)が押されていなければX、押されていればYの操作になります。
- ※3 LTRT（AB/LTRT）スワップモードの設定に応じて入れ替えます。同時に押された場合は、増加側を有線します。

## 設定モード
デバイスのB(BOOTSEL)ボタンを3秒ほど押し続けると、LEDの点滅が変化して「設定モード」に入ります。

- もう一度B(BOOTSEL)ボタンを押すと、設定モードを終了します。
- ネジコンをねじって、その状態でAボタンを押すと、そこがねじりの最大値になります。
- 上下キーで中央のデッドゾーン（スティックの遊び値）の削減量の調整ができます。32段階で、大きいほどデッドゾーンが減ります（最大にするとまっすぐ走れなくなります）。初期値は16です。
- アクセル・ブレーキに使う、II（アナログ）ボタン、またはL（アナログ）ボタンを押すと、アナログの感度調整ができます。4段階に切り替え可能で、大きいほど鈍く（Aカーブがきつく）なります。ボタンを押すと、LEDがその段階分だけ点滅します。
- STARTボタンで、すべての設定値の初期化ができます。
