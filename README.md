# 俺はneGconをForza Horizon 6でつかいたいんじゃ for RP2040<BR>(neGconXboxPICO)

オリジナルは[neGcon2SwitchPICO](https://github.com/v9938/neGcon2SwitchPICO)です。

これのファームウェアを、Xbox 360 Controller互換になるよう書き換えたものです。

## Firmware書き込み方法

デバイスのB(BOOTSEL)ボタンを押しながら、デバイスのリセットボタンを押すか、USBケーブルを接続します。
PC側にUSBメモリとして現れるドライブに、neGcon2Xbox_pico_*.uf2をコピーします。

## メタモード
何も押していない状態でSTARTボタンを押すと、メタモードに入ります。

- そのままSTARTボタンを離すと、メニューボタンをタップしたものとして扱います。
- STARTボタンを押している間、以下のアサインになります。

|neGconボタン | Xboxコントローラ | 備考 |
| :- |  :- |  :- |
| ねじり (アナログ) | 右アナログスティックのX/Y | ※2 |
| I (アナログ) |  | ねじり(アナログ)の機能切り替え。 |
| II (アナログ) | 左アナログスティックのY減少側 | ※3 |
| L (アナログ) | 左アナログスティックのY増加側 | ※3 |
| 十字キー上 | Yボタン | |
| 十字キー下 | XBOXボタン | |
| 十字キー左 | LBボタン | |
| 十字キー右 | RBボタン | |
| R (デジタル) | ※1 | |
| A (デジタル) | 右スティック押し込み | |
| B (デジタル) | 左スティック押し込み | |

- ※1: メニュー+Rボタン。Steamのコントローラ設定モードで、ここに好きな操作をアサインできます。
- ※2: I (アナログ)が押されていなければX、押されていればYの操作になります。
- ※3 LTRT（AB/LTRT）スワップモードの設定に応じて入れ替えます。同時に押された場合は、増加側を有線します。

## 設定モード
デバイスのB(BOOTSEL)ボタンを3秒間押し続けると、LEDの点滅が変化して「設定モード」に入ります。
もう一度B(BOOTSEL)ボタンを押すと、設定モードを終了します。

|neGconボタン | Xboxコントローラ | 備考 |
| :- |  :- |  :- |
| ねじり (アナログ) | 左アナログスティックのXの最大値指定 | ※1 |
| II (アナログ) | II (アナログ)ボタンの感度設定 | ※2 |
| L (アナログ) | L (アナログ)の感度設定 | ※2 |
| 十字キー上 | ねじり (アナログ)のデッドゾーン削減量拡大 | ※3 |
| 十字キー下 | ねじり (アナログ)のデッドゾーン削減量縮小 | ※3 |
| A (デジタル) | ねじり (アナログ)の最大値確定 | ※1 |
| START | ※4 | |

- ※1: ネジコンをねじって、その状態でAボタンを押すと、そこがねじり（ハンドル）の最大値になります。
- ※2:- アクセル・ブレーキに使うアナログボタンの感度調整ができます。4段階に切り替え可能で、大きいほど鈍く（Aカーブがきつく）なります。ボタンを押すと、LEDがその段階分だけ点滅します。
- ※3: ハンドル中央のデッドゾーン（スティックの遊び値）の削減量の調整ができます。32段階で、大きいほどデッドゾーンが減ります（最大にするとまっすぐ走れなくなります）。初期値は16です。
- ※4: 短押しでUSB MSCモードに移行します。3秒間の長押しで、ファクトリーリセットを行います。

## USB MSCモード
設定モードでSTARTボタンを押すと、設定モードの設定値をUSBメモリ上のファイルとして読み書きできるモードになります。

LEDが「水色で高速点滅」の間は、USBメモリとしての認識中です。認識が完了すると、「水色でゆっくり点滅」に変化します。
もう一度B(BOOTSEL)ボタンを押すと、USB MSCモードを終了します。

### ファイルの読み書き

USBメモリとして認識されると、「CONFIG.INI」というファイルが現れます。このファイルを保存することで、設定のバックアップが可能です。またこのファイルを変更することで、設定の更新を行うことができます。

ファイルの読み込みアクセス中は、LEDが青色に変化します。また書き込みアクセス中は、赤色に変化します。赤色の間は、USBケーブルの抜き差しを行わないでください。

### 起動しない！

USBメモリの内容が破損すると、デバイス接続時にLEDが紫色で停止し、コントローラとして使用不可能になります。この状態になってしまったら、5回連続でリセットすることで、ファクトリーリセットを行います。

## 再ビルドについて

再ビルドの際は、Arduino IDEを以下の設定にする必要があります。

| 項目 | 設定値 |
| :- |  :- |
| ボード | Raspberry Pi Pico |
| C++ Exceptions | Disabled |
| Flash Size | 2MB(Sketch: 1MB, FS: 1MB) |
| CPU Speed | 133MHz |
| IP/Bluetooth Stack | IPv4 Only |
| Optimize | Small (-Os) |
| RTTI | Disabled |
| Stack Protector | Disabled |
| USB Stack | No USB |
